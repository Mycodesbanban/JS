o que e SSH

E uma forma segura de conectar sevidores sem precisar usa sua senha toda vez


para gerar a senha no nosso computador usamos:
 
 primeiro vamos entra na pasta SSH

 cd ~/.ssh/

 depois rodamos o seguinte comando

 ssh-keygen -t ed25519 -C "(email de exemplo)"
